## Actions principales
- Visualisez, recherchez, filtrez vos torrents et téléchargements depuis le dashboard
- Ajoutez un torrent (magnet ou fichier) via le bouton dédié
- Supprimez, copiez, streamez, ou affichez les infos détaillées d’un torrent
- Utilisez le scraper pour rechercher et injecter des liens externes
- Gérez en masse (bulk delete, bulk unrestrict)
- Rafraîchissez les liens morts si besoin

## File d’attente & priorités
- Accédez à la page « File d’attente » pour voir, prioriser, mettre en pause ou supprimer des jobs
- Modifiez la priorité avec les boutons ▲/▼, supprimez ou relancez un job
- La file se synchronise automatiquement toutes les 5 s

## Feedbacks & sécurité
- Les notifications, loaders et messages d’erreur s’affichent en temps réel
- Le token Real-Debrid n’est jamais exposé côté client
