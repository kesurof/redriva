# Guide d’utilisation Redriva

- Accédez à l’interface web (http://localhost:3000 ou 5173 selon la stack)
- Connectez-vous avec votre token Real-Debrid (stocké côté serveur)
- Visualisez, recherchez, filtrez vos torrents et téléchargements
- Ajoutez un torrent (magnet ou fichier)
- Supprimez, copiez, streamez, ou obtenez les infos détaillées d’un torrent
- Utilisez le scraper pour rechercher et injecter des liens externes
- Gérez en masse (bulk delete, bulk unrestrict)
- Rafraîchissez les liens morts si besoin

Voir la documentation technique pour les endpoints API et les cas d’usage avancés.
