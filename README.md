# Projet Redriva – Gestionnaire Real-Debrid

Ce dépôt contient la structure complète et tous les fichiers nécessaires pour lancer une application de gestion Real-Debrid moderne, sécurisée et évolutive.

## Structure du projet

- `src/` : code source frontend (UI, composants, pages)
- `backend/` : code source backend (API, modèles, sécurité)
- `docs/` : documentation technique, guides, FAQ
- `tests/` : tests unitaires, intégration, E2E
- `scripts/` : scripts utilitaires, maintenance
- `config/` : fichiers de configuration
- `logs/` : logs applicatifs
- `data/` : données persistantes, base de données
- `systemd/` : fichiers d’unités systemd pour le service

## Lancement rapide

Voir `docs/INSTALL.md` pour l’installation et le lancement complet.
